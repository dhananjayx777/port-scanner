# 🔍 Port Scanner

A simple and fast TCP Port Scanner built with Python. It scans a target IP address or domain, detects open ports, identifies common services,
and supports optional banner grabbing.

## ✨ Features

- Fast multithreaded scanning
- Scan IP or domain
- Custom port range
- Banner grabbing
- Common service detection
- Easy command-line interface

## 🚀 Usage

```bash
python main.py <target>
```

Example:

```bash
python main.py scanme.nmap.org
```

## 🛠 Built With

- Python
- Socket Programming
- Multithreading
